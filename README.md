# Sprint 2-Front — Sorriso Novo

Evolução do site da Sprint 1 para a Sprint 2, mantendo o layout original e adicionando:
- Responsividade (mobile, tablet e desktop)
- Acessibilidade (texto alternativo, navegação por teclado, skip link)
- JavaScript para interações e validação de formulários
- Organização de pastas em `assets/`

## Páginas
- index.html (Início)
- sobre.html (Sobre o projeto)
- integrantes.html (Equipe com RM e links)
- servicos.html (O que oferecemos)
- noticias.html (Atualizações)
- faq.html (Perguntas frequentes)
- contato.html (Formulário)

## Como testar
Abra `index.html` no navegador.

## Versionamento
Crie o repositório **Sprint-2-Front** e faça os commits frequentes durante a evolução.
